## Simpla.Laravel

https://laravel.com

INSTALL

- install Laravel Homestead https://laravel.com/docs/6.x/homestead
- ssh vagrant
- cd project directory
- clone project https://github.com/simpladevru/simpla-laravel
- cp .env.example .env
- composer install
- php artisan key:generate
- php artisan migrate
- npm install
- npm run dev
